# Support

Thanks for your interest in ControlPlane contracts and tooling.

## Where to Ask Questions

- **GitHub Discussions**: Q&A, design proposals, and integration questions.
- **GitHub Issues**: Bugs or actionable feature requests.

## Security Issues

Please use the private reporting instructions in [SECURITY.md](./SECURITY.md). Do not file public issues for vulnerabilities.

## Maintainer Response

Maintainers respond on a best-effort basis. For urgent issues, include as much detail as possible to speed up triage.

## Organizational Responsibility

Organizations implementing these contracts and tools are responsible for their own deployment governance, change management, and operational oversight.
